    Gerby

    The website for the Stacks project uses Gerby as the underlying system. It was written by Pieter Belmans and Raymond Cheng. It is an open source system, which combines

    • a LaTeX processing tool built using plastex
    • a website built using Flask
    We mostly modified the existing Stacks Project website to make this one! A lot of the content (except the math) is therefore from the Stacks Project, with some modifications.
