    Overview of the API

    This page is mostly unmodified from the Stacks Project's API page.

    It is possible to query this project yourself through an API. This way you don't have to scrape the information from the HTML pages and it is consistent with our goal that the content of this project be as open as possible (thanks, Stacks Project!).

    You can contact the creators of the API at stacks.project@gmail.com.

    Statements

    There are two different possibilities to get the statement of a tag:

    1. HTML, without proof: 
      https://fppf.site/ega/data/tag/‹tag›/content/statement

      example: HTML statement for tag 015I

    2. HTML, with proof: 
      https://fppf.site/ega/data/tag/‹tag›/content/full

      example: HTML statement (with proof) for tag 015I

    Because the mathematics is parsed by MathJax, the HTML output contains raw LaTeX math. It is up to the user to handle this (e.g. either use MathJax, or some image generation tool). {% endblock %} {% block sidebar %} {{ macros.singlePageLinks() }} {% endblock %}
